## Development & Building

### Building and Packaging

Build the project:

```bash
pnpm build
```

GZip the output:

```bash
pnpm package
```

### Development

Install the dependencies.

```bash
pnpm i
```

Start the developtment server:

```bash
pnpm dev
```
